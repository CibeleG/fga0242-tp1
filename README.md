# Pré-requisito

Para rodar esse projeto é necessário ter instalado:

# Execução dos testes (via Maven)

```
   mvn test
```

# Execução dos testes (via Intellij)

## 1. Passo
Clique em editar configurações

## 2. Passo
Escreva "test" embaixo do run e clique em Apply e OK.

## 3. Passo
